<h1 align="center">🌸 Sakura Beats – Music Web Player</h1>

<p align="center">
A modern anime-themed music player built using <b>HTML, CSS, and JavaScript</b>
</p>

<hr>

<h2>🚀 Features</h2>
<ul>
  <li>🎵 Play, Pause, Next, Previous controls</li>
  <li>⏱ Real-time progress bar with seek functionality</li>
  <li>📋 Dynamic song queue system</li>
  <li>🔍 Instant song search</li>
  <li>🖼 Album cover, artist & lyrics display</li>
  <li>🎨 Aesthetic anime UI with blur effects</li>
  <li>🔁 Smooth view transitions</li>
</ul>

<hr>

<h2>🛠 Tech Stack</h2>
<ul>
  <li><b>HTML</b> – Structure</li>
  <li><b>CSS</b> – Styling & animations</li>
  <li><b>JavaScript</b> – Logic & functionality</li>
</ul>

<hr>

<hr>

<h2>⚙️ How It Works</h2>
<ul>
  <li>Songs are stored as objects in JavaScript</li>
  <li>Dynamic UI rendering using DOM manipulation</li>
  <li>Audio handled via HTML audio element</li>
  <li>Event listeners manage playback & progress</li>
  <li>Queue updates automatically</li>
</ul>

<hr>

<h2>▶️ How to Run</h2>
<ol>
  <li>Clone or download the project</li>
  <li>Keep media files inside <b>pulse</b> folder</li>
  <li>Open <b>index.html</b> in browser</li>
</ol>
